# dental-schedule-app
A dental scheduling app that will let you access any hygienists and their office schedule. 

## Built with
Backend:
  - Core Java (1.8)
  - Spring MVC
  - JPA
  - MariaDB
  - Apache TomCat
  - EclipseLink 2.5
  
Frontend:
  - HTML5
  - Vanilla JS
  - JQuery
  - Bootstrap
  - Wow.js
